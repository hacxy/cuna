# Cuna

终端交互式中英语言翻译工具.

```
$ cuna --help
cuna/0.0.0

Usage:
  $ cuna

Commands:
        打开翻译文本输入框
  clip  从剪切板中读取内容直接翻译

For more info, run any command with the `--help` flag:
  $ cuna --help
  $ cuna clip --help

Options:
  -v, --version  Display version number
  -h, --help     Display this message
```
